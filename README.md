## Module

API Validation & Reports

## Features

* Attendance Management APIs
* Leave Management APIs
* Request Validation using express-validator
* PostgreSQL Integration
* SQL Reports
* RESTful API Design

## Technologies Used

* Node.js
* Express.js
* PostgreSQL
* express-validator
* dotenv
* cors

## API Endpoints

### Attendance

POST /api/attendance/add

GET /api/attendance

### Leave

POST /api/leaves/apply

GET /api/leaves

## Installation

1. Clone repository
2. Run npm install
3. Configure PostgreSQL
4. Update db.js
5. Run node app.js
